History of JS

It was created in 1995 to act as a scripting language for basic animation and small interactions within the Netscape browser. Netscape and Sun Micro Systems originally developed JS to be a companion to Java. At the time, Java was used to develop larger back end processes and JS primarily handled the front end aspects. Since JavaScript was owned by Sun Micro Systems, the newly formed standard was called ECMAScript which became supported by all major browers. These days JavaScript and ECMAScript are one in the same thing.

First JS and now NodeJS

NodeJS released in 2009 is similar in some repects to JavaScript but without the use of a browser. Used for backend APIs, networking features, and single page applications, along with automation tasks.
